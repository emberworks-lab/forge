# web (React / Next / Astro / Vue / Svelte / etc.) interview

## Ask about

- **State management** — Zustand / Redux / TanStack Query / React Context / Pinia / Svelte stores.
- **API layer** — REST / GraphQL / tRPC.
- **Component library** — none / shadcn / Material / Chakra / custom.
- **Routing** — Next App Router / TanStack Router / React Router / Nuxt / SvelteKit / file-based.
- **Testing approach** — RTL / Vitest / Playwright / Cypress.

## Generate

- `kit-create-feature.md` — component + state + types + tests.
- `kit-add-route.md` — page + loader + nav update.
- `kit-add-api-route.md` — server handler + types.
- `kit-deploy.md` — build + deploy for the chosen hosting target.
